## Getting Started

### Run on Linux / macOS
1. Download the latest release from the [Releases page](../../releases).
2. Open a terminal and navigate to your download folder:
```bash
   cd ~/Downloads
```
3. Make the file executable:
```bash
   chmod +x Game
```
4. Run the game:
```bash
   ./Game
```
