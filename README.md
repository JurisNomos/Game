# Game

[繁體中文](./README.zh-tw.md)

A simple terminal-based math game built with C# where players answer randomly generated math questions and earn points for correct answers.

## Features
- Menu-based operation selection
- Minimum of 5 questions per game
- Score tracking system
- Game history stored in memory
- View previous game results

## How to Run
1. Ensure you have dotnet environment installed.
   
   ```bash
   dotnet --version
   ```

2. Clone or Download this repository.

   ```bash
   git clone
   ```

3. Navigate to the project folder.
   
   ```bash
   cd Game/
   ```

4. Run the project.

   ```bash
   dotnet run
   ```
