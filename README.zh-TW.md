# 遊戲

[English](./README.md)

一個使用 C# 開發的簡單終端機數學遊戲，玩家需要回答隨機產生的數學題目，答對即可獲得分數。

# 功能特色
- 選單式選擇
- 每場遊戲至少 5 題
- 分數計算系統
- 遊戲紀錄儲存在記憶體中
- 可查看過往遊戲紀錄

# 執行方式
1. 請先確認已安裝 dotnet 環境。
   
   ```bash
   dotnet --version
   ```

2. 下載或複製此專案。
   
   ```bash
   git clone git@github.com:JurisNomos/Game.git
   ```

3. 進入專案資料夾。
   
   ```bash
   cd Game/
   ```

4. 執行專案。

   ```bash
   dotnet run
   ```
